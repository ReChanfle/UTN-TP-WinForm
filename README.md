🛒 Catalog Management Application
📌 Overview

This project is a desktop application designed to manage items in a commercial catalog. It is built to be generic and adaptable to any type of business.

The application allows full administration of catalog items, while the stored data can later be consumed by external services such as websites, e-commerce platforms, mobile applications, or digital/printed catalogs.

⚠️ External integrations are not part of this project.

🚀 Features
- List all items
- Search items by different criteria
- Add new items
- Edit existing items
- Delete items
- View item details
- Item Structure

Each item contains:

- Code
- Name
- Description
- Brand (dropdown selection)
- Category (dropdown selection)
- Images (one or more, no limit)
- Price
- Additional Management
- Manage Brands
- Manage Categories
- Support for multiple images per item
  
🗄️ Database

The application uses an existing SQL Server database provided with the project.
